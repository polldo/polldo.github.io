Tiny Snake Board

I made this tiny gaming board to approach the world of embedded systems.

The code has been written in Assembly from scratch, using my own drivers, for didactic purpose.

It is based on Attiny85 microcontroller. I had strict constraints due to the low amount of SRAM available in the Attiny85: 512 Bytes.

I really loved building this project and I'm glad that it's still working with the same battery.

Dolp - gaming library for Arduino

Dolp is a gaming library for Arduino boards. It allows to develop simple games in a very easy way.

This short video shows off a multiplayer pong game made with the library, it leverages Bluetooth Low Energy for the communication part.

Dolp is not complete but I work on it whenever i have some time. So, it is still under development but it can already be used with Arduino nano33ble boards. However it hasn't been spread to the arduino community yet, it needs some love before being released.

  • possibility to draw arrows and square marks
